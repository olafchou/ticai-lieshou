# Stock Catalyst Hunter Examples

这些示例更适合 GitHub 浏览和快速理解 skill 用法。

## 文字输入示例 1

**输入**

```text
这条新闻利好哪些A股：美加墨世界杯赛程出炉，揭幕战将于6月12日3点举行。
```

**输出重点**

- 体育题材
- 足球周边
- 啤酒消费联想

## 文字输入示例 2

**输入**

```text
这消息对应什么题材股：中东冲突升级，国际油价盘中拉升。
```

**输出重点**

- 油气
- 航运
- 黄金避险

## 图片输入示例 1

**输入**

```text
帮我看看这张图会影响哪些A股股票。
```

**适用场景**

- 财经截图
- 快讯截图
- 群聊转述
- 海报或新闻卡片

## 图片输入示例 2

**输入**

```text
这张截图会带动哪些A股？帮我直接找相关题材和个股。
```

**输出重点**

- 先提取图片里的事件
- 再找受影响方向
- 最后映射相关个股
